    • Initialize another new disk

    If you don't find your new disk on the file explorer, then it need to be initialised. Initialising the disk requires that the disk should not be formatted before.


    Steps to initialise a new disk on Windows


    To make disk ready for Windows use, initialise it by following general steps:
    1. Right-click the disk > tap 'Initialize Disk'.
    2. To continue with default style, click 'ok'.
    3. Right-click unallocated space then tap 'New Simple Volume'.
    4. Tap 'Next', confirm volume size.
    5. Tap 'Next', follow on-screen prompts.
    6. Lastly, tap 'Finish'.

    Note:
    1. Be cautious that chosen disk is selected in the Initialize Disk dialog box.
    2. Also, the disk should be online, not offline.
