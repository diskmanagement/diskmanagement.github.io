Disk Management



Disk management is one of the 4 management functionality under operating system. With Disk management, the utmost benefit a user gets is that his drive gets sorted in a better way. This provides you with features to arrange data in various drives - such as optical drives, flash drives & hard disk drives. You can easily assign drive letter, format it and alter disk volume as well. It can shrink or extend disk volume.



What are the functions performed in disk management?


With its availability in various windows versions (Windows 11, Windows 10, Windows 8, Windows 7, Windows Vista, Windows XP, and Windows 2000) it performs various functions such as:

  • Drive partition
  • Change in drive letter
  • Drive formatting
  • Initiate another new drive
  • Drive partition Contraction
  • Drive partition Expansion
  • Change in file system of a drive
  • Drive partition deletion
  • Troubleshoot various disk issues


How can I access/open disk management in Windows?


  • Go to Control Panel > System & Security.
  • Tap 'Administrative Tools'.
  • Double-click Computer Management.
  • Under 'Storage', select 'Disk Management'.

    How can I assign a drive letter via Disk Management?

    1. Visit Disk Management as an administrator >Choose the drive.
    2. Tap 'Change Drive Letter and Paths'.
    3. Click 'Change', choose new drive letter.
    4. Tap 'OK'.

    Why I cannot change Drive letter of C drive?

    This is because Drive letter of that partition which contains windows OS can't be altered. And C drive contains it, hence it cannot be done.
